# TWYS JOB
This application runs as a *job* to scrape data from www.investing.com once a day and as a email sender for administration bussiness

## Environment
Please setting the environment first.
```
# TWYS settings
TWYS_HOST
TWYS_TOKEN

# Gmail Settings
GMAIL_EMAIL
GMAIL_PASSWORD
```
